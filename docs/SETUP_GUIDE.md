# 校园网自动化监控 - 使用指南

## 🎯 新方案说明

这个重构方案用**后台监控脚本**替代手动操作，具体包括：

### 功能特性
- ✅ **自动检测校园网状态**：每60秒检查一次校园网连接
- ✅ **校园网断开时**：自动禁用有线网适配器（避免手动操作）
- ✅ **每天早上7点**：自动启用有线网并运行登录脚本
- ✅ **持续监控**：后台运行，无需每天手动启动
- ✅ **日志记录**：所有操作都记录在 `network_monitor.log` 中

---

## 🚀 快速开始

### 方案 A: 简单方案（推荐新手）
1. 手动运行一次以测试：
   ```
   右键点击 start_monitor.bat → 以管理员身份运行
   ```

2. 如果正常运行，可以添加到启动文件夹（开机自启）：
   ```
   快捷键 Win+R → 输入 shell:startup → 回车
   将 start_monitor.bat 快捷方式放入此文件夹
   ```

### 方案 B: 任务计划程序（推荐）- 更可靠的开机自启

1. **打开任务计划程序**
   - 快捷键：`Win+R` → 输入 `taskschd.msc` → 回车

2. **创建新任务**
   - 右键点击"任务计划程序库" → "创建基本任务"
   - 名称：`NJUPT Campus Network Monitor`
   - 描述：`Automatically manage NJUPT campus network connection`

3. **配置触发器**
   - 选择"计算机启动时"

4. **配置操作**
   - 操作：选择"启动程序"
   - 程序或脚本：`start_monitor.bat` 的完整路径
     ```
     C:\njupt-login\start_monitor.bat
     ```
   - 起始于：`C:\njupt-login`

5. **配置条件**
   - 勾选"如果计算机处于闲置状态，则等待"取消选中
   - 勾选"唤醒计算机以运行此任务"

6. **完成并启用**
   - 完成创建
   - 右键点击任务 → 属性 → "常规"标签
   - 勾选"不管用户是否登录都要运行"

## 📊 工作流程详解

```
启动监控脚本
    ↓
└─ 每60秒检查一次 ─┐
    ├→ 校园网在线？
    │    ├ 是 → 确保有线网适配器启用
    │    └ 否 → 禁用有线网适配器（减少困扰）
    │
    └→ 是否是早上7点？
         └ 是 → 启用有线网 + 运行登录脚本
```

## ⚙️ 自定义配置

编辑 `network_monitor.py` 的以下变量：

```python
ADAPTER_NAME = "linenet"        # 网络适配器名称
CHECK_INTERVAL = 60             # 检查间隔（秒）
MORNING_HOUR = 7                # 早上启用时间（小时）
CAMPUS_NETWORK_URL = "https://p.njupt.edu.cn:802/"  # 校园网地址
```

所有操作都记录在 `network_monitor.log` 文件中：
```

[2024-01-15 12:30:45] 校园网已断开，正在禁用有线网适配器...

```


## 🔧 故障排除

### 1. "需要管理员权限"错误
- 确保以管理员身份运行 `start_monitor.bat`
- 对于任务计划程序，勾选"不管用户是否登录都要运行"

### 2. 网络适配器操作失败
- 确认适配器名称正确（使用 `ipconfig /all` 查看）
- 检查是否有其他应用锁定网络适配器

### 3. 登录脚本未执行
- 确认 `main.py` 和配置文件存在且正确
- 查看 `network_monitor.log` 了解具体错误

### 4. 早上没有自动启用
- 检查系统时间是否准确
- 修改 `MORNING_HOUR` 变量到合适的时间

## 💡 进阶用法

### 使用 nssm 创建 Windows 服务（可选）

如果想让脚本像真正的 Windows 服务一样运行：

1. 下载 [nssm](https://nssm.cc/download)
2. 放到某个目录，比如 `C:\nssm`
3. 在命令行中运行：
   ```
   nssm install NJUPTNetworkMonitor C:\njupt-login\network_monitor.py
   nssm start NJUPTNetworkMonitor
   ```

优点：崩溃自动重启，更加稳定
